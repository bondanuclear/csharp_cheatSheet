# Use of LINQ
## LINQ - Language-Integrated Query - являє собою мову запитів до джерела даних. Існує декілька видів LINQ, наприклад: 
- LINQ to Objects
- LINQ to Entities
- LINQ to XML
- LINQ to DataSet
- Parallel LINQ (PLINQ)
